What is the Document Object Model (DOM)

The Document Object Model (DOM) is a programming interface that allows scripts to interact with and manipulate web documents.

How do you select an element from the DOM?

There are many ways to select DOM in the Javascript like using getElementById, getElementsByClassName, getElementsByTagName, querySelector.

What is event delegation in the context of dom and why it is useful?

The process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.It allows you to attach a single listener to the parent container.

How do you manupulate an element's attributes and style using DOM?

Using setAttributes() Method. The setAttributes has two string parameters. The first one is attribute name and the second is attribute value.
